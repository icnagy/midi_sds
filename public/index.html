Midi SDS transfer tool

Oh hi,

1. Upload your samples

Drop your samples here

2. Transfer your samples

Press the buttons in low-to-high key order and wait for the transfer to complete.

3. Set root note and keyspan

When the transfer is finished set the base note and the keyspan on the DPM. When finished, press EXEC to save the settings then EXEC again to save the sample name. Then press the next sample transfer button.

If you want to do a multisampled patch/drumkit set the keyspan to be less than A0>G9. In that case the next transfered sample will be allocated for the same patch. For more information please refer to the DPM V3 owener's manual page 6.5 step 12.
